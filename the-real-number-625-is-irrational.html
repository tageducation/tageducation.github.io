29. the real number √625 is irrational true or false?​


Answer:

False.

Step-by-step explanation:

The [tex]\sqrt{625}[/tex] is [tex]25[/tex] which can be expressed in ratio of [tex]\frac{1}{25}[/tex] thus it is rational.

A number can only be called irrational if it cannot be expressed in any ratio.

[tex]\pi[/tex] for example is irrational because it's decimal is not repeating nor terminating.
