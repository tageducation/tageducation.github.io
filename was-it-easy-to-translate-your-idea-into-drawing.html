3. Was it easy to translate your idea into drawing?​


Yes, I think it is an easy work to translate our ideas into draws.

Drawings in the form of branches are commonly used to formulate complex concepts into easy-to-understand forms. Making ideas in the form of a draw is called mind mapping. Mind mapping is a way of grouping several ideas in the form of a structured framework to help remember or analyze a problem. Mind mapping is also referred to as the process of transferring thought forms in the brain into written and draw forms. While a mind map is an image of a frame or mind map that you have grouped based on the ideas in your mind.

Drawings in the form of branches are commonly used to formulate complex concepts into easy-to-understand forms. This is easy because we simply do it with the following steps:

1. Determine the main topic

The topic can come from the problem we want to solve, and is usually in the form of key words consisting of 1 to 3 words.We can start by choosing a topic that has a broad meaning, such as communication.

2. Add branches by main topic

Next, we need to look for ideas that are most closely related to the main topic and make them the first branch of that topic. Nonverbal and verbal are types of communication. So, if we are still having trouble determining the main branch, try to identify it by type or other sections.

3. Break down the main branch into several sub-topics

After getting the idea as the main branch, we can break it down into several sub-topics. We can try to define some ideas with a smaller scope. So that it can become a branch of the main branch.

4. Connect each idea with a line

After determining several sub-topics, don't forget to connect each idea with a line. By using lines, it will be easier for us to understand the contents of the mind map that we create. Starting from the main topic to the most specific sub-topics.

5. Use color coding on the connecting line in each sub-topic

To make it easier to remember each idea in a sub-topic, we can add color codes to connecting lines or images. We can see in the example image below, each topic has its own color code, for example 'gesture' which has a purple color.

6. Develop sub-topics with the most important ideas

We will certainly find a lot of ideas that may have something to do with sub-topics. However, to make an effective mind mapping, we need to determine the ideas that have the closest connection with the sub-topic. This is useful to prevent you from entering ideas that you don't need.

7. Add a small note if needed

Not infrequently, later the mind mapping that you make can have a large number of branches. So that you don't forget the meaning of each idea, there's nothing wrong with making small notes. In addition, these notes can also be a description of ideas that are considered difficult or complex.
