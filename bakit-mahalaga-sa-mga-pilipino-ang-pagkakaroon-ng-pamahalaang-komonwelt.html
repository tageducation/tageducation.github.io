Kahalagahan ng Commonwealth

Narito ang mga dahilan kung bakit mahalaga para sa mga Pilipino ang pagtatayo ng isang Commonwealth:

Ang Commonwealth ay ang unang pamahalaan sa ilalim ng mga Amerikano kung saan ang namumuno ay isang Pilipino (Manuel L. Quezon). Ang Commonwealth ay ang magiging batayan ng mga Amerikano upang tignan ang kakayahan ng mga Pilipino sa pamumuno ng isang bansa. Ang Commonwealth ang magiging daan upang lumaya ang Pilipinas mula sa kamay ng mga Amerikano.

Explanation:

Ang pamahalaang Commonwealth ang opisyal na pamahalaan ng Pilipinas mula 1935 hanggang 1946, ngunit hindi kabilang dito ang mga taon kung saan naka-exile ang pamahalaan sa Amerika noong sakupin ng mga Hapon ang bansa. Pinalitan nito ang Pamahalaang Insular ng Amerika, at naisakatuparan sa pamamagitan ng Batas Tydings-McDuffie.
