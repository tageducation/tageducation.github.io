What is a combination lock called?

Almost all combination locks should actually be known as permutation locks because the order of the numbers that must be entered cannot be switched

Is combination lock of a vault permutation or combination?

Famous joke for the difference is: A “combination lock” should really be called a “permutation lock”. The order you put in the numbers of lock matters. For example, a true “combination lock” would accept both 17–01–24 and 24–17–01 as correct.

Why is it called a combination lock?

It is simply the combination of characters that let us pick holes in the term. Many cheap combination locks are in-fact combination locks and not permutation locks. For some the order does matter, but for others the order does not matter. Both are called combination locks in English.

22. A lock contains 5 digits in its code. how many codes can be formed if all digits must be different?


Answer:

There are 30, 240 possible codes that can be formed in a lock with five digits.

Further Explanation:

This problem can be solved using the idea of the Fundamental Concepts of Counting. The Fundamental Principle of Counting tells us that if the first event has [tex]p[/tex] possible outcomes and the second event has [tex]q[/tex] possible events, then the possible outcomes when the first and second event happen simultaneously is [tex]p\times{q}[/tex] ways.

Some Counting Techniques

1. Permutation

2. Combination

3. Factorial

In this situation, there are two possible conditions that we may consider.

Two Conditions To Consider

1. When repetition is not permitted

2. When repetition is permitted

Most of the locks are having a numeric combination. This means that our choices are from 1, 2, 3, 4, 5, 6, 7, 8, 9, 0. In other words, we have ten numeric values to choose from.

Solution:

                 _1st__   _2nd_   _3rd_   _4th_   _5th_

There is ten possible choices that we can place in the first position located from the left.

Since repetition is not permitted, the number of choices that we can place in the second position is 9.

The number of choices that we can put on the third position is 8.

The number of choices that we can put on the fourth position is 7.

The number of choices that we can put on the fifth position is 6.

Then multiply all the number of choices placed on each position (Fundamental Principle of Counting Concept).

                                  [tex]10\times{9}\times{8}\times{7}\times{6} = 30,240[/tex]

This means that the lock can have 30, 240 possible codes using the numerical values provided that there's no value that is repeated.
