LINE SEGMENT

Line segment [tex]\overline{\rm mn}[/tex] is

[tex]\blackdot[/tex][tex]A\bullet---------------\bullet B[/tex]

What is a Line Segment?

A line segment is a measurable route between two places. Line segments can make up the sides of any polygon because they have a set length. The line segment AB is depicted in the diagram below, and its length is defined as the separation between its ends, A and B.

Line Segment Symbol

The line segment sign, a bar at the top, represents a line segment. It is written as [tex]\overline{\rm AB}[/tex]

Line Segment Formula

We know the coordinates of the two endpoints, let's look at how to get the length of a line segment. We apply the distance formula in this instance, which is

D = √[(x2 - x1)² + (y2-y1)²]

Where, (x1, y1) and (x2, y2) are the coordinates of the given points.

Difference Between Line, Line Segment, and Ray

A line cannot be measured since it has inescapable ends.A line segment can be measured since it has a start point and an end point.Line segments constitute the sides of any polygon since they have a set length.A ray cannot be measured since it has only one start point and no end point.The sun's rays are a good illustration of how to understand the concept of rays because they have a starting point but no finish.
