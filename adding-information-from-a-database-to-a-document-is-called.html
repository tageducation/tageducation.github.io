1. Adding information from a database to a document is called _____.


Document-oriented database

An adding of information from a database to a document is called a Document-oriented database. The Document-oriented database is an exact kind of database that works on the standard of dealing with 'documents’ rather than firmly defined tables of information.  

The document-oriented database plays a significant role is gathering data from documents and getting them into a searchable, prearranged form.  

In an interactive database system you must define a schema before adding accounts to a database. The schema is the arrangement described in a formal language supported by the database.

Examples:Suppose one document has:Two namesOne addressList of ages of a home’s occupants. A second document might have:Four namesTwo addressesNo age information

A document-oriented database will take the data in both and store them according to type, able to handle non-fixed length data sets.
