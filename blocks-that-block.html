7. a pile of blocks has 60 blocks in the bottom row, 54 blocks in the second row, 48 blocks in the third row, and so on until there is only 6 blocks on the too row. How many blocks are there in all?​


Answer:

There are 10 blocks.

Step-by-step explanation:

Given:   [tex]a_1=60[/tex],   [tex]a_n=6[/tex],   [tex]d=-6[/tex]

Find:   [tex]n=?[/tex]

Formula:   [tex]a_n=a_1+(n-1)d\\[/tex]

Substitute the given in the formula.

[tex]a_n=a_1+(n-1)d\\6=60-6(n-1)\\6=60-6n+6\\6n=60+6-6\\6n=60\\\frac{6n}{6}=\frac{60}{6}\\n=10[/tex]

29. A pile of blocks has 40 blocks in the bottom row,36 blocks in the second row,32 blocks on the third row and so on until there are only 4 blocks on the top row.How many blocks are there in the 7th row? How many blocks are there in all?


There are 16 blocks in the 7-th row

There are 220 blocks in all

[tex]\texttt{ }[/tex]

Further explanation

This problem is about Arithmetic Progression.

[tex]40 , 36 , 32 , 28 , 24 , ... , 4[/tex]

where:

a = 40

d = 36 - 40 = -4

Let's find the number of blocks in the 7th row:

[tex]\boxed{T_n = a + (n-1)d}[/tex]

[tex]T_7 = 40 + (7-1)(-4)[/tex]

[tex]T_7 = 40 + 6(-4)[/tex]

[tex]T_7 = 40 - 24[/tex]

[tex]T_7 = 16[/tex]

[tex]\texttt{ }[/tex]

Let's find the total number of blocks:

[tex]\boxed{T_n = a + (n-1)d}[/tex]

[tex]4 = 40 + ( n - 1 )(-4)[/tex]

[tex]4 - 40 = ( n - 1 )(-4)[/tex]

[tex]-36 = ( n - 1 )(-4)[/tex]

[tex]-36 \div -4 = ( n - 1 )[/tex]

[tex]9 = n - 1[/tex]

[tex]n = 9 + 1[/tex]

[tex]n = 10[/tex]

[tex]\texttt{ }[/tex]

[tex]\boxed{S_n = \frac{1}{2}n ( 2a + (n-1)d )}[/tex]

[tex]S_{10} = \frac{1}{2}(10) ( 2(40) + (10-1)(-4) )}[/tex]

[tex]S_{10} = \frac{1}{2}(10) ( 80 - 36 )}[/tex]

[tex]S_{10} = \frac{1}{2}(10) ( 44 )}[/tex]

[tex]S_{10} = 220[/tex]
