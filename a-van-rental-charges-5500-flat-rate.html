2. A van rental charges P5,500.00 flat rate for a whole-day tour in CALABARZON of 5 passengers and each additional passenger added P500.00 to tour fare. Express a piecewise to show to represent the van rental in terms number of passenger n.​


Solving the problem of linear equations of one variable and the inequality of one variable.What is a linear equation?

A linear equation is an equation that has a minimum of a ranked variable with the highest degree of the variable being one. Why is it called linear? Because the relationship of this mathematical relationship is depicted with a straight line in a cartesian coordinate system.

One Type of Linear Equations in One Variable Linear Equation

As the name implies, a one-variable linear equation contains only one variable of rank one.

The general forms of the One Variable Linear Equation are:

ax + b = 0

where:

a = coefficient

b = constant

x = variable

One-variable linear inequality?

A single-variable linear inequality is a form of inequality having a single variable or modifier in which this inequality is usually represented by symbols <, >, ≤ and ≥. Inequality is always marked with a hyphen < to express less than, > to express more ≤ than or less than or equal to, as well as ≥ to express not less than or more than or equal to.

The Case: A van rental charges P5,500.00 flat rate for a whole-day tour in CALABARZON of 5 passengers and each additional passenger added P500.00 to the tour fare. Express a piecewise to show to represent the van rental in terms of a number of passengers n.​

Given: Van rental charges for 5 passengers is P5500.

Find: Express a piecewise function to show to represent the van rental in terms number of passengers n.

Solution:

given that, van rental charges for 5 passengers is P5500.

So,

A number of passengers = less than or equal to 5.

Total charge = P5500 .

Then, 5500 for n ≤ 5.

Now, given that, each additional passenger added P500 to the tour fare.

and, Total passengers are n.
Total charge = charges for 5 passengers + charges for more than 5 members with each paying P500.
Total passengers = n.

Therefore, total charge = 5500 + 500(n - 5) where n > 5.

Hence, required piecewise function of total charge = 5500 + 500(n - 5) where n > 5.

=> 5500 + 500n - 2500 where n > 5 .

=> 3000 + 500n where n > 5. (Ans.)

So, piecewise functions are:-

5500 n ≤ 5.3000 + 500n n > 5.
