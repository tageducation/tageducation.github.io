Answer:

para hindi mamatay agad ung itatanim

Explanation:

Hope it helps

Sagot:

Paghahanda ng Lupa o (Soil preparation)

Ang lupa ay isang kalihim para sa lahat ng elementong kailangan ng mga halaman para lumaki: mga sustansya, organikong bagay, hangin, at tubig. Ang lupa ay nagbibigay din ng suporta para sa mga ugat ng halaman. Kapag maayos na inihanda at inaalagaan, ang lupa ay maaaring mapabuti bawat taon at patuloy na magpapalago ng mga halaman magpakailanman.

Dagdag Kaalaman:

Bakit mahalagang ihanda ang lupa bago itanim?

Kinokontrol ay pag hahanda sa bukid ang mga damo, nire-recycle ang mga sustansya ng halaman, at nagbibigay ng malambot na masa ng lupa para sa paglipat at isang angkop na ibabaw ng lupa para sa direktang pagtatanim.

Bakit tayo naghahanda ng lupa?

 Paghahanda ng lupa: Inihanda ang lupa bago itanim ang mga buto. Ang lupa ay lumuwag upang madagdagan ang pagsipsip ng tubig at mga pataba. Ang pagluwag ng mga particle ng lupa ay nagdaragdag ng humus at mga sustansya sa lupa na nagpapataas ng mga ani ng pananim.

Paano mo inihahanda ang lupa bago itanim?

Ang pagdaragdag ng organikong bagay sa anyo ng compost at lumang pataba, o paggamit ng mulch o lumalagong mga pananim na takip (green manures), ay ang pinakamahusay na paraan upang ihanda ang lupa para sa pagtatanim. Ang pagdaragdag ng mga kemikal na pataba ay maglalagay lamang ng ilang mga sustansya at wala nang magagawa para sa pagpapanatili ng mabuti, marupok na lupa.
