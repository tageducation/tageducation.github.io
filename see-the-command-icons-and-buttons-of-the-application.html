3. describe and give function the following commands,button or icon1.border button2.formula bar3.table style button4.resize icon5split tables button6.table button ​


Answer:

Border button: This is a button that allows you to add a border around a table or a group of cells. It can be used to separate different sections of a spreadsheet or to make the data more organized.

Formula bar: This is a bar that appears at the top of your screen when you are working with a spreadsheet. It allows you to enter formulas, which are instructions that tell the computer what to do with the data in the spreadsheet.

Table style button: This button allows you to change the appearance of your table by applying different styles. These styles can change the colors, font, and other formatting options.

Resize icon: This icon allows you to change the size of the table or cells by clicking and dragging the edges.

Split tables button: This button allows you to split a table into multiple smaller tables. It can be useful when you want to organize your data in a more manageable way.

Table button: This button allows you to create a new table in your spreadsheet. It can also be used to insert or delete rows and columns in an existing table.
