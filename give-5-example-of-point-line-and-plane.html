30. DEFINE THE FOLLOWING 1)POINT 2) LINE 3) PLANE 4) COLLINEAR POINTS 5) COPLANAR POINTS NOTE GIVE EXAMPLE IN EACH ITEM​


Answer:

Point: A point is a basic unit of geometry that has no size or shape. It is represented by a dot and is used to define the location of an object in space. Example: The location of a city on a map can be represented by a single point.

Line: A line is a set of points that extends infinitely in both directions. It has no thickness or width and is defined by two points on it. Example: A line can be drawn to represent the path of a moving object on a graph.

Plane: A plane is a flat, two-dimensional surface that extends infinitely in all directions. It has no thickness and can be represented by three non-collinear points. Example: A piece of paper can be considered a plane.

Collinear Points: Collinear points are three or more points that lie on the same line. Example: The points (1,2), (2,3), and (3,4) are collinear.

Coplanar Points: Coplanar points are three or more points that lie in the same plane. Example: The points (1,2,3), (4,5,6), and (7,8,9) are coplanar.

1. Point: A point is an undefined term in geometry that represents a location in space and has no size or dimension. It is represented by a dot and is represented by a capital letter. For example, A, B, C.

2. Line: A line is a straight path that extends infinitely in both directions and has no width or thickness. It is represented by two points and can be represented by a letter in between two points. For example, AB, CD.

3. Plane: A plane is a flat surface that extends infinitely in all directions. It has length and width but no thickness. It can be represented by three non-collinear points or a letter symbol representing the plane. For example, ABC or plane P.

4. Collinear Points: Collinear points are points that lie on the same line. For example, A, B, and C are collinear points if they lie on the same line AB.

5. Coplanar Points: Coplanar points are points that lie on the same plane. For example, A, B, and C are coplanar points if they lie on plane ABC.
