This involves adding integers with like and unlike signs.

Integers with like signs can be added by adding the numbers directly and including the sign.

Integers with unlike signs can be added by:

subtracting the absolute value of the number with the lesser absolute value from the absolute value of the number with the greater absolute value, and including the sign of the number with the greater absolute value.

Note that the absolute value of a number is its distance from zero in a real number line.
