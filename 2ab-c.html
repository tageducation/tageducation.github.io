28. a=4 b=-2 c=6 find the value of the ff.1. c(2a+b)2. 3a-2b3. a b c4. ab/c5. (2a-b) (2a+b)​


Answer:

1) 36

2) 16

3) -48

4) -4/3 or -1.333...

5) 60

Step-by-step explanation:

To find the value of the given algebraic expressions, all we have to do is to substitute our variable values into the unknowns and evaluate the resulting expression.

Let n be the value of the algebraic expression without unknown variables.

Item 1:

n = c(2a+b)

n = (6) [2(4) + (-2)]

n = 6(8-2)

n = 48 - 12

n = 36

Item 2:

n = 3a - 2b

n = 3(4) - 2(-2)

n = 12 + 4

n = 16

Item 3:

n = abc

n = (4)(-2)(6)

n = -48

Item 4:

n = ab / c

n = (4)(-2) / 6

n = -8/6

n = -4/3 or -1.333...

Item 5:

n = (2a-b) (2a+b)

n = [2(4)-(-2)] [2(4)+(-2)]

n = (8+2) (8-2)

n = (10) (6)

n = 60

In similar problems, don't be discouraged to answer or solve for the asked just because it seems complicated. They are oftentimes easy: You just have to substitute and evaluate.
