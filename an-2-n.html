Answer:

To find (h/g)(n), you need to first evaluate h(n) and g(n) and then divide h(n) by g(n).

h(n) = 2n - 2

g(n) = n^2 + 2n

Therefore, (h/g)(n) = (2n - 2) / (n^2 + 2n)

This simplifies to:

(h/g)(n) = (2n - 2) / (n^2 + 2n) = (2n - 2) / (n(n+2))

To simplify further, you can factor the denominator to get:

(h/g)(n) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2)) = (2n - 2) / (n(n+2))

This expression cannot be simplified further, so the answer is (h/g)(n) = (2n - 2) / (n(n+2)). The correct answer is therefore (D) 2n-2 / n^2+2n.
