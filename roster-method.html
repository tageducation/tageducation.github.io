3. what is roster method and rule method​


Answer:

(1) Roster method or Listing method : In this method a set is described by listing elements, separated by commas, within braces { }. ... (2) Set - builder method or Rule method : In this method, a set is described by a characterizing property P(x) of its elements x.

The roster method is defined as a way to show the elements of a set by listing the elements inside of brackets. An example of the roster method is to write the set of numbers from 1 to 10 as {1,2,3,4,5,6,7,8,9 and 10}.

The rule method is where a set is described by a characterizing property P(x) of its elements x. In such a case the set is described by {x : P(x) holds} or {x | P(x) holds}, which is read as 'the set of all x such that P(x) holds'.

Examples:

Seasons of the year

Roster Method

{Summer, Spring, Fall, Winter}

Rule Method

{ x | x is a season of the year}

15. what is the difference between roster method and rule method


Answer:

Difference Between Roster Method and Rule Method

Roster method and rule method are the two methods used in representation of a set.

What is roster method?

Roster method is the listing of the elements in a set within the pair of brackets { } and are separated by commas.

The elements can be listed in any order but elements must not be repeated.

Examples:

W is the set of last four months of the year.

W = {September, October, November, December}

The set of all odd numbers less than 9.

X = {1, 3, 5, 7}

The set F, consists of all the positive multiples of 5 that are less than 50.

F = {5, 10, 15, 20, 25, 30, 35, 40, 45}

The set of all vowels of the English alphabet.

V = {a, e, i, o, u}

The set of all letters in the word BEAUTIFUL.

Z = {B, E, A, U, T, I, F, L}

What is rule method?

Rule method is specifying the rule or property or statement describing a set by indicating the properties that it's members must satisfy.

The element of the set is described by using a symbol ‘x’ or any other variable followed by a colon (:) or this symbol |. The symbol stands for "such that". Then, the property of the set is enclose in braces.

Examples:

W is the set of last four months of the year.

W = {x : x is one of the last four months of the year}

The set of all odd numbers less than 9.

X = {x | x is an odd number less than 9}

The set F, consists of all the positive multiples of 5 that are less than 50.

F = {x | x is positive multiple of 5 that is less than 50}

The set of all vowels of the English alphabet.

V = {x : x is a vowel in English alphabet}

The set of all letters in the word BEAUTIFUL.

Z = {x : x is a letter in the word BEAUTIFUL}
