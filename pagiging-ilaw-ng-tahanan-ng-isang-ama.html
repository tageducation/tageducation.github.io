Ilaw ng tahanan

Haligi ng tahanan

Ang haligi ng tahanan ay ating mga ama. Mga lalaking nagpalaki, nagmahal at nagbigay ng ating mga pangangailangan.

14. Kung ang ang ina ay ilaw ng tahanan ano naman ang ama at anak/ikaw??​


Ang kahalagahan ng bawat meyembro ng pamilya  Ama Ina Anak

Ang ama ang haligi ng tahanan, bilang haligi ng tahanan siya ang dapat magbigay ng pangunahing pangangailangan ng kanyang asawa at mga anak. Siya ang naghahanap  buhay upang maibigay ang mga nais ng kanyang mga pamilya. Hindi biro ang maging haligi ng tahanan kailangan mong maging matatag, matapang at masipag upang mabuhay ng marangal ang iyong pamilya.kaya naman napakalaking responsibilidad ng isang ama.  Ang anak ang kasayahan ng tahanan, ang mga supling ng isang pamilya ang siyang inspirasyon ng isang ama at ina, sila ang dahilan kung bakit lumalaban at patuloy na nagsisikap an gating mga magulang, kaya bilang anak tungkulin natin na sumunod sa ating mga magulang, kailangan natin sila ng igalang mahalin at pahalagahan. Ang ina ang ilaw ng tahanan, bilang isang ina ng tahanan tungkulin niyang alagaan pagsibihan ang kanyang asawa at mga anak, ang kamay ng isang ina ang hahaplos at mag aalaga sa bawat meyembro ng pamilya, masarap makaranas ng pag aalaga at pag mamahal ng isang ina walang makakapantay ang kanyang sakripisyo upang sa lahat ng oras ay maibigay niya ang kanyang libreng serbisyo sa bawat meyembro ng pamilya.

Ang mahalaga ay magawa ng ama ang obligasyon o pananagutan ng mga ilaw ng tahanan. 21. kung ng ina ay ang ilaw ng tahanan ano naman ang ama?


Answer:

Ang Ama ay ang Haligi ng Tahanan.

#CarryOnLearning

Answer:

ang AMA ay ang Haligi ng Tahanan

Explanation:

Ang ama ay tinagurian bilang haligi ng ating mga tahanan sapagkat siya ang tumatayong proteksyon ng kanyang buong pamilya. Ang proteksyon ay inaasahang manggagaling sa ama ng tahanan dahil sa paniniwalang ang mga kalalakihan ay ang may kapasidad na prumotekta gamit ang kanyang pisikal na kalakasan. Kung ano man ang maranasan ng kanyang pamilya, ang ama ang handang gumawa ng paraan upang malagpasan nila ito at hindi niya hahayaang mapahamak ang kanyang pamilya.  

Ang ama ng tahanan din ang siyang nagbibigay ng pinansyal na pangangailangan ng kanyang pamilya. Noong unang mga panahon, lalake lamang ang may karapatang maghanap-buhay o magtrabaho. Ngunit sa modernong panahon ngayon, maging ang ina ng mga tahanan ay inaasahan narin na maghanap-buhay upang matustusan ang kanilang pinansyal na pangangailangan sa pang araw-araw.
