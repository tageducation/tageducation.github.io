Solve by Factoring

: x² + 10x + 9 = 0

Step 1:

Consider the form x² + bx + c. Find a pair of integers whose product is c and whose sum is b. In this case, whose product is 9 and whose sum is 10.

1,9

Step 2:

Write the factored form using these integers.

( x + 1 )( x + 9 ) = 0

x + 1 = 0

x + 9 = 0

Step 3:

Set x + 1 equal to 0 and solve for x

Set x + 1 equal to 0

x + 1 = 0

Subtract 1 from both sides of the equation.

x + 1 - 1 = 0 - 1

x = -1

Set x + 9 equal to 0 and solve for x

Set x + 9 equal to 0

x + 9 = 0

Subtract 9 from both sides of the equation.

x + 9 - 9 = 0 - 9

x = -9

THE FINAL ANSWER IS

x = -1, -9.
