7. Find the length of the longest diagonal in parallelogram FAST 3×-1,5.5,2×+7?​


Answer:

The length of the longest diagonal in a parallelogram is 24.86.

Step-by-step explanation:

1. First, the shape of the half of a parallelogram is a right triangle with sides given [tex]3x - 1, 5.5, 2x + 7[/tex]

where:

[tex]a = 3x - 1[/tex]

[tex]b = 5.5[/tex]

[tex]c = 2x + 7[/tex]

2. Using Pythagorean theorem, we substitute the values of a, b and c.

[tex]c^{2} = a^{2} + b^{2}[/tex]

[tex](3x-1)^{2} = 5.5^{2} + (2x + 7)^{2} \\[/tex]

[tex]9x^{2} - 6x + 1 = 30.25 + 4x^{2}  + 28x + 49[/tex]

Using transposition to combine like terms, we get:

[tex]5x^{2}  - 34x - 78.25 = 0[/tex]

3. Using quadratic formula, solve for x:

[tex]x = \frac{- b ± \sqrt{b^{2} - 4ac } }{2a}[/tex]

[tex]x = \frac{34 ± \sqrt{34^{2} - 4(5)(-78.25) } }{2(5)}[/tex]

[tex]x = \frac{34 ± \sqrt{1156 + 1565} }{10}[/tex]

[tex]x = \frac{34 + \sqrt{2721} }{10}[/tex]

[tex]x = \frac{34±52.16}{10}[/tex]

[tex]x = 8.62  and - 1.82[/tex]

4. Therefore we use x = 8.62 to find the longest diagonal.

5. Substitute the value of x to 3x - 1  and 2x + 7 we get:

[tex]3x - 1[/tex]

[tex]3(8.62) = 24.86[/tex]

[tex]2x + 7[/tex]

[tex]2(8.62) = 24.24[/tex]

6. Therefore, since 24.86 is greater than 24.24, we take the higher value  which is 24.86 to be the longest diagonal in  the parallelogram.

7. Note: Using 2x + 7 to be the hypotenuse and 5.5 & 3x - 1 as the legs of the triangle will produce a length of 20.58 only.

Keywords: Parallelogram, Pythagorean theorem, Quadratic formula

20. the sides of a parallelogram and an included angle are 8m, 13m and 120 respectively, find the length of the shorter diagonal​


Step-by-step explanation:

we can find the length of the shorter diagonal using the formula:

Length of shorter diagonal = √(long side squared + short side squared - 2 * long side * short side * cos Θ)

Where:

Long side = 13 meters

Short side = 8 meters

Included angle = 120 degrees

Each pair of angles in the parallelogram equals 180 degrees. Since we want the shorter diagonal, we need to subtract 120 from 180 getting 60 degrees, or Θ = 60 degrees.

Length of diagonal

= √(13^2 + 8^2 - 2 * 12 * 8 * cos 60)

Length of the shorter diagonal

=11.70 m
