Answer:

We know that [tex]4 + 3 =7[/tex] is also true when we say that [tex]3+4=7[/tex]. But when we say [tex]4-3=1[/tex] is not the same as [tex]3-4=-1[/tex], this is because the operation subtraction is not commutative. Subtraction is not commutative under the set of real numbers. This means to say that interchanging the values will make the result different.

Step-by-step explanation:

The explanation for this problem can be based on the basic properties of the set of real numbers in general. Remember that the set of real numbers are everything that you can see everywhere such as temperature, money, distances, population, and so on. These are numbers that do exist in reality. There are numbers that don't exist in reality, we call them "imaginary numbers".

Some Basic Properties of Real NumbersClosure Property. The set of real numbers are closed under the following operations, addition, multiplication, and subtraction. This means to say that whenever we use any of these operations, the result is still part of the set of real numbers. That's what we meant by the word closure. Commutative Property The set of real numbers are commutative under the operation multiplication and addition. This means that using either addition or multiplication, interchanging the values doesn't make the result different. For example, [tex]4+3=7=3+4[/tex] and [tex]5\times 7=35 =7 \times 5[/tex].Associative Property. The set of real numbers are associative under the operation multiplication and addition. This means to say that the groupings do not change the answer using either of the mentioned operations. For example, [tex]3+(4+5)=12=(3+4)+5[/tex] and [tex]3\times (4\times 5)=60=(3 \times 4)\times 5[/tex].

The reason why “the sum of x and y” is equal to “the sum of y and x,” but “the difference of x and y” is not the same as “the difference of y and x" is very simple.
