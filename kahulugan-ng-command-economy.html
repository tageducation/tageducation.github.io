What is a Command Economy?

A command economy is a system where the government, rather than the free market, determines what goods should be produced, how much should be produced, and the price at which the goods are offered for sale. It also determines investments and incomes. The command economy is a key feature of any communist society. Cuba, North Korea, and the former Soviet Union are examples of countries that have command economies, while China maintained a command economy for decades before transitioning to a mixed economy that features both communistic and capitalistic elements.

9. ano ang kahulugan ng command economy


Answer:

Kahulugan ng Command Economy.

Ang Command Economy ay ang uri ng ekonomiya na kung saan ang lahat ng pagpapasya kung paano sasagutin ang tatlong tanong na kinakaharap ng bawat ekonomiya ay nagmumula sa gobyerno (central administration).

Explanation:

Ang tatlong tanong na sasagutin ng bawat uri ekonomiya (all types of economy):  

Ano ang mga produktong gagawin?Paano ito gagawin?Para kanino ang mga gagawing produkto?

Hindi madaling pag-isipan kung ano ang produktong bibigyang pansin ng gobyerno.

Kabilang sa mga ito ang mga sumusunod:

Serbisyo sa mamamayan (polis, guro, doktor at iba pa)Pwersang militar Agrikultura Industriya (technical).

Mahalagang malaman ng pamahalaan ang kailangan at kagustuhan ng mga mamamayan bago magpasya kung alin ang pagtutuunan.

Sasagutin din ng pamahalaan kung paano isasagawa ang manufacturing ng produkto. Gagamit ba ng human labor o automation? Anong mga buwis ang ipapatong sa negosyo? Ilan alamang ito sa maraming tanong na mahalagang pag-isipan bago malaman kung paano isasagawa ang produksyon.

Matapos mabuo ang produkto, para kanino ito? Narito ang ilan sa mga katanungang pagbabasihan ng sagot:

Ano ang average income ng mga mamimili? Ang kultura ng mga mamimiliEdadUso o kalakaran at iba pa.

 Mga katangian ng Command Economy:

May central authority na sasagot sa mga tanong na may kinalaman sa ekonomiyaAng awtoridad ang gumagawa ng lahat ng desisyonAng mga indibidwal ay may maliit o walang papel sa mga desisyong pang ekonomiya.

Mga kalakasan ng Command Economy (advantages):

Madaling palitan ang uri o direksyon ng ekonomiyamay kasiguruhan ang kinabukasan ng mga mamamayan mura ang mga serbisyong pambubliko.  

Dahil iisa ang nasususnod sa command economy, katulong ang buong puwersa ng pamahalaan, madali nitong naipapatupad ang mga pagbabago. Hindi ito nakaasa sa free market na may sariling direksyon.

Sa Command Economy ay nabibigyan ng trabaho ang bawat isa sa mga mamamayan nito kaya’t nakasisisguro ang lahat na mayroong silang papel na ginagampanan at dahil dito ay nakakaasa sila sa mga benepisyo na nagmumula sa gobyerno.

Isa sa pangunahing tungkulin ng Command Economy na alagaan ang kapakanan ng mga mamamayan, at ang pagpapatibay ng social services para sa lahat ang nakikitang sagot dito. Ang lahat ay gagamutin kung magkakasakit, makakapag-aral ng mura o walang bayad, at may rasyon na mga pagkain. Mahalagang banggitin natin na ang kalidad ng serbisyo ay kadalasang mababa kung ihahambing sa Market Economy.

Mga kakulangan o kahinaan ng Command Economy (disadvantages):

Napipigilan ang creativity ng mga nasasakupanWalang kalayaan ang mga indibiduwal na kuhanin ang kanilang mga pinapangarap, hindi layunin ang individual happiness.Sa laki ng operasyong nakapatong sa pamahalaan, mabagal ang serbisyo.

Dahil ibinibigay ng pamahalaan ang lahat ng pangangailangan (needs) ng mamamayan, nawawala ang “incentive” ng indibiduwal na lumikha para sa sarili. namamatay ang creativity sa loob ng Command Economy. Ang karaniwang nakikita sa ibang uri ng ekonomiya na paghanap ng individual happiness ay nalilimitahan sa loob nito dahil ang pangangailangan ng estado ang binibigyan ng halaga.  

Ang Command Economy ay karaniwang makikita sa mga bansang Communist gaya ng North Korea na pinaka magandang halimbawa nito.  

Maihahambing natin ang bansang sumusunod sa Command Economy sa isang Ant colony. Dito ay may reyna, sundalo at mga manggagawa ng may kanya kanyang papel na ginagampanan.

13. Halimbawa ng Command Economy at Mixed Economy


Sistema ng ekonomiya ng command

Ang sistema ng command economy ay isang sistema ng estado na kumokontrol at namamahala sa lahat ng mga aksyong pang-ekonomiya ng bawat mamamayan nito at ang mga entidad ng negosyo nito ay direktang pagmamay-ari ng gobyerno.

Mga Katangian ng Command Economy System

Ang bawat sistemang pang-ekonomiya ay dapat magkaroon ng sarili nitong batayang konsepto ng pag-iisip. Ang konseptong iyon ay bubuo kung paano gumagana ang sistema sa pagsasaayos ng aktibidad sa ekonomiya.

Ang mga sumusunod ay ang mga katangian ng isang command economy system:

• Ang pamahalaan ang pangunahing may hawak ng kontrol at regulasyon ng lahat ng aktibidad sa ekonomiya

• Ang mga komunidad ay walang karapatan sa mga karapatan sa pagmamay-ari sa mga mapagkukunang pang-ekonomiya

• Kinokontrol at sinusubaybayan ng gobyerno ang presyo ng mga bilihin at ang rate ng interes sa merkado

• Ang mga komunidad ay may mga limitasyon sa pagsasagawa ng mga aktibidad na pang-ekonomiya

• Kinokontrol ng gobyerno ang lahat ng problema sa ekonomiya, tulad ng kahirapan, inflation, kawalan ng trabaho, at iba pa

Halimbawa ng isang Command Economy System:

• Ang North Korea na hanggang ngayon ay sumusunod pa rin sa isang command economy system, ito ay pinalalakas ng lahat ng North Korean economic activities ay dapat makakuha ng permiso mula kay President Kim Jong Un.

• Ang isa pang bansa na gumagamit ng command economy system ay ang Cuba, kung saan ang pamahalaan ay gumagawa ng sentralisadong pagpaplano upang ang produksyon at pamamahagi ng mga pangunahing pangangailangan ay pinamamahalaan ng pamahalaan, sa pangunguna ni Pangulong Miguel Mario Diaz.

Mixed sistema ng ekonomiya

Ang mixed economic system ay isang sistemang pang-ekonomiya kung saan ang gobyerno at pribadong sektor ay nagtutulungan sa isa't isa sa pagpapatakbo ng ekonomiya sa isang bansa.

Tinatawag itong mixed economic system dahil ito ay kombinasyon ng Liberal (kapitalista) na sistemang pang-ekonomiya at Command (sentralisadong) sistemang pang-ekonomiya. Sa pagsasagawa, ang sistemang pang-ekonomiya na ito ay nagpapahintulot sa pribadong sektor na magsagawa ng mga aktibidad na pang-ekonomiya ngunit mayroong interbensyon mula sa pamahalaan sa bansang kinauukulan.

Mga Katangian ng Mixed Economic System

Makikilala natin ang sistemang pang-ekonomiya na ito mula sa mga katangiang taglay nito. Ang mga katangian ng sistemang pang-ekonomiya na ito ay ang mga sumusunod:

• Ang gobyerno at pribadong sektor ay gumaganap ng aktibong papel at nagtutulungan sa ekonomiya.

• Binibigyan ng gobyerno ng kalayaan ang pribadong sektor na magsagawa ng mga gawaing pang-ekonomiya. Gayunpaman, ang gobyerno ay nagpapataw ng mga limitasyon at maaari ring makialam.

• Gumagawa ang pamahalaan ng mga plano, regulasyon, at patakarang may kaugnayan sa ekonomiya.

• Nagaganap ang kumpetisyon sa merkado sa loob ng makatwiran at malinis na mga limitasyon kung saan nakikilahok ang pamahalaan sa pagsubaybay.

• Tutukuyin ng mekanismo ng pamilihan ang uri at dami ng mga produktong ginawa.

• Kinokontrol ng pamahalaan ang lahat ng mahahalagang mapagkukunan na may kaugnayan sa kabuhayan ng mga tao.

• Pareho ang papel ng gobyerno at pribadong sektor sa aktibidad ng ekonomiya.

Mga halimbawa ng mga bansang may pinaghalong sistema ng ekonomiya :

Ang mga bansang sumusunod sa magkahalong sistemang pang-ekonomiya ay nakalista bilang mga dating hindi nakahanay na bansa, kung saan ang karamihan ay nasa mga kontinente ng Asia at Africa. Sila ay Indonesia, India, Malaysia, Egypt.

20. kahalagahan at kapakinabangan ng Traditional Economy, Market Economy, Command Economy, Mixed Economy​


Answer:

Ang traditional economy ay ang pinakapayak at pinakamatandang sistema sa apat na uri. Ang sistema na ito ay nakabatay sa paglikha ng produkto at serbisyo na sumusunod sa naaakmang panahon. Madalas ang sistema na ito ay makikita sa mga rural na lugar kung saan ang agrikultura ang pangunahing kabuhayan.

Ang market economy ay kumikilos sa konsepto ng free market. Ang pamahalaan ay hindi nakikialam sa ekonomiya. Ang pamahalaan ay may maliit na kontrol lamang sa mga pinagkukunang yaman at hindi nakikiaalam sa mga mahalagang bahagi ng produksyon. Ang regulasyon ng pamilihan ay nagmumula sa mga mamimili at sa epekto ng paggalaw ng supply at demand.

Sa isang command economy, malaking bahagi ng ekonomiya ay kontrolado ng isang sentralisadong gobyerno. Ito ay tinatawag din na “planned economy”.

Dahil sentro ang pamahalaan ng sistema na ito, ang pamahalaan ay bahagi ng pagpaplano hanggang sa pamamahagi ng mga pinagkukunang yaman. Ang command economy ay may kakayahan na lumikha ng maayos na supply ng mga pinagkukunang yaman at nakakatulong din ito para mabigyan ang mga mamamayan ng mababang presyo ng bilihin. Madalas ay kontrolado ng gobyerno ang mga mahahalagang industriya sa bansa tulad ng tubig, kuryente, paliparan, daungan at mga riles. Ang mga mamamayan ay may kontrol sa mga hindi gaanong importanteng sektor ng ekonomiya tulad ng agrikultura.

Pinagsasama sa mixed economy ang mga katangian na taglay ng command economy at market economy, ito ang rason kung bakit tinatawag din itong dual systems.

Karamihan ng mga bansa sa daigdig ay sumusunod sa mixed economy. Kung saan ang karamihan ng mga industriya ay nasa pribadong pamamalakad at ang natitira ay bumubuo sa mga pampublikong serbisyo na nasa ilalim ng pamamalakad ng pamahalaan.

23. Bigyan kahulugan ang mga sumusunod: 1. Traditional Economy- 2. Command Economy 3. Mixed Economy 4. Market Economy


Answer:

Market Economy

            Sa market economy, ang kasagutan sa pangunahing katanungang pang-ekonomikoya ginagabayan ng mekanismo ng malayang pamilihan. Sa gantong sistema, ang bawat kalahok – konsyumer at prodyuser, ay kumikilos alinsunod sa kanilang pansariling interes na makakuha ng malaking pakinabang. Ang market economy ay nagpapahintulot sa pribadong pagmamay-ari ng kapital, pakikipag-ugnayan sa pamamagitan ng presyo, at nangangasiwa ng mga gawain.

Command Economy

               Sa command economy, ang ekonomiya ay nasa ilalim ng komprehensibong kontrol at regulasyon ng pamahalaan. Ang pagkontrol ay alinsunod sa isang planong nauukol sa pagsusulong ng ekonomiya na pinangangasiwaan mismo ng sentralisadong ahensya (cental planning agencies).

               Ang mga patakaran sa command economy  ay ipinatupad sa dating Soviet Union. Sa kasalukuyan, nananatiling may ganitong sistemang pang-ekonomiya ang Cuba at North Korea.

Mixed Economy

             Ang mixed economy ay isang sistema na kinapapalooban ng elemento ng market economy at commmand economy. Walang maituturing na isang kahulugan ang mixed economy. Ito ay kinapapalooban ng magkaugnay na katangian ng dalawang sistema tulad ng malayang pakikilahok sa mga gawaing pangkabuhayan na pinahihintulutan ng pamahalaan at pagkontrol ng pamahalaan sa ilang gawaing pangkabuhayan. Ang salitang mixed economy ay nilikha upang tukuyin ang isang sistemang nabuo at may mga katangian bunga ng pagsasanib o kombinasyon ng command  at market economy.

             Ang sistemang mixed economy ay nagpapahintulot din na makagawa ng pribadong pagpapasya ang mga kompanya at indibidwal .

30. Kaibahan ng command economy sa mixed economy​


Answer:

Command vs Mixed Economy

Sa Command Economy, Ang responsable sa paggawa o pagbuo ng desisyon ay ang gobyerno. Ang mga indibidwal ay walang pakealam sa gagawing desisyon sapagkat hindi isinasa alang-alang ang kanilang sasabihin. Ang Command Economy ay ginagamit ng mga komunistang bansa gaya ng Rusya, Tsina, Cuba at North Korea.

Habang sa Mixed economy,

Parehong indibidwal at gobyerno ay kasama sa pagbuo ng desisyon para sa ekonomiya ng bansa. Ito ay ang kombinasyon ng Market at Command Economy. Ginagamit ito ng mga demokratikong bansa tulad ng Pilipinas, Estados Unidos, France at iba pa.
