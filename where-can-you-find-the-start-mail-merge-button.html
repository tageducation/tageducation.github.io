22. Write the word True on the line if the statement iscorrect, else write False.1. The Mail Merge allows you to produceMultiple copies of a document at onceThat can be sent to various recipients.2. When using Mail Merge, type in theRecipient's information individually onThe document3. The Mail Merge button can be accessedFrom the insert ribbon.​


1. True

The Mail Merge feature in Microsoft Word allows you to produce multiple copies of a document that are personalized for different recipients. This can be useful for creating mass mailings such as letters, emails, or labels.

2. False

When using Mail Merge, you don't have to type in the recipient's information individually on the document. Instead, you create a data source, such as a spreadsheet or a database, that contains all of the recipient information, and then you use that data source to populate the document.

3. False

The Mail Merge button is not located in the insert ribbon, It can be accessed from the Mailings tab on the ribbon in Microsoft Word.
