6. It is an example of software that can be used to create electronic collage.


Answer[ C ]  Picture Collage Maker

There are a lot of better picture collage plug-ins for Photoshop but the Picture Collage Maker is the best for non-artist students because it is free and user-friendly for beginners. It is also less taxing on the computer RAM, unlike Photoshop which requires large amounts of RAM for processing information. The final result of Picture Collage Maker is Amateurish in design but it gets the job done.

As a Graphic Design Artist, I suggest using this as a student to avoid getting overloaded with homeworks. Adobe Photoshop may have an intimidating User Interface for beginners and requires time to completely utilize all its tool to the fullest. Nonetheless, you can start to train on Adobe Photoshop if you plan to enter the industry of Arts.
