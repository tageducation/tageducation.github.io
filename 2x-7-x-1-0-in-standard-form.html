Answer:

the answer is 2x² + 5x - 7 = 0

Step-by-step explanation:

FOIL METHOD

(2x+7)(x-1) = 02x (x-1) = 2x² - 2x7 (x-1) = 7x - 7

combine like terms, perform operations and rearrange.

2x² + ((-2x)+(7x)) - 7 =02x² + (5x) - 7 = 02x² + 5x - 7 = 0
