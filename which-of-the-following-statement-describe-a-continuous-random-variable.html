Discrete random variable, namely a random variable defined in a discrete sample space that can only undergo a finite number of different values (as many as integers).Continuous random variable, that is a random variable defined in a continuous sample space that can undergo every value in an interval (infinitely many).

The mathematical model that relates each value of a random variable with the probability that it occurs in the sample space is called a probability distribution.
