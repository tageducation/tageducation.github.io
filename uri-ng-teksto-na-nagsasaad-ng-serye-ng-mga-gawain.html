1. Uri ng teksto na nagsasaad ng serye ng mga gawain


Uri ng teksto na nagsasaad ng serye ng mga gawain ay tekstong prosidyural.

Mga detalye ng paliwanag:

Ang teksto ng prosyidural ay isang teksto na naglalaman ng mga tagubilin kung paano gawin o gawin ang isang bagay. Ang layunin ng teksto ng pamamaraan ay magbigay ng mga tagubilin para sa paggawa o paggawa ng isang bagay.

Ang mga katangian ng procedure text ay:

Naglalaman ng mga puntos na karaniwang isinusulat ng mga numeroGamit ang isang command verb, halimbawa: Fold... Watch...Gumamit ng mga aktibong pandiwaIsinulat nang sunud-sunod at sistematikong ayon sa mga hakbang at pagkakasunod-sunod

Ang mga halimbawa ng mga teksto ng prosyidural ay mga teksto kung paano i-restart ang computer nang maayos, mga teksto kung paano gumawa ng mga handicraft. Kasama rin sa mga recipe ng pagkain ang procedure text.
