Definition of inverse function

An inverse function is a mapping that has the opposite direction to the function. For example, suppose a function maps from a set A to B. So, what is meant by an inverse function is a function that maps from B to A. .

A function with certain properties has an inverse, the function is a function that has a bijective property or a one-to-one correspondence. Likewise with function composition. The composition of two functions that have an inverse will also have an inverse.

Determining the Inverse Function Formula

To determine the formula for the inverse function of the function f, the following steps can be performed:

Let f(x) = y

Expressing x in y

Determine the formula for f^(-1) (x) by remembering f^(-1) (y) = x and replacing the variable y with x.

Example 1

Given f: R → R where f(x) = 2x - 5. Find f^(-1) (x)!

Solution:

Let f(x) = y

= f(x) = 2x - 5 becomes y = 2x - 5

Determine the formula for f^(-1) (x) by remembering f^(-1) (y) = x and replacing the variable y with x

= y = 2x - 5

= 2x = y + 5

= x = (y + 5)/2

= f^(-1) (x) = (y + 5)/2

Example 2

If f(x) = (2x + 1)/(x - 4) Find f^(-1) (x)!

Solution

Let f(x) = y

= f(x) = (2x + 1)/(x - 4) becomes y = (2x + 1)/(x - 4)

Determine the formula for f^(-1) (x) by remembering f^(-1) (y) = x and replacing the variable y with x

= y = (2x + 1)/(x - 4)

= y(x - 4) = 2x + 1

= yx - 4y = 2x + 1

= yx - 2x = 4y +1

= x(y -2) = 4y + 1

= x = (4y + 1)/(y - 2)

= f^(-1) (x) = (4x + 1)/(x - 2)

Solution
