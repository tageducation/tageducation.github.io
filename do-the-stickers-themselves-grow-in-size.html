10. zoe collect stickers, she can arrange those 15 or 18 stickers to a page without any leftover what is the least number of sticker for which you can do this ​


Answer:

90

Step-by-step explanation:

To find the least number of stickers that Zoe can arrange to a page without any leftover, we need to find the smallest common multiple (LCM) of 15 and 18.

First, we can find the prime factorization of both numbers:

15 = 3 x 5

18 = 2 x 3 x 3

Then, we can take the union of the prime factors, but we need to make sure to include each factor the maximum number of times it appears in either factorization:

LCM = 2 x 3 x 3 x 5 = 90

Therefore, the least number of stickers that Zoe can arrange to a page without any leftover is 90.
