2. Ano ang kahulugan ng Paglilingkod ng pampribado


Answer:

Ang sektor ng paglilingkod ay ang nasa likod ng paggawa o produksyon, at maging sa pag konsumo ng mga produkto sa merkado. Mayroong dalawang uri ng paglilingkod, ang paglilingkod na pampribado at paglilingkod na pampubliko.

Ang paglilingkod na pampribado ay ang lahat ng paglilingkod na nagmumula sa pribadong sektor. Ibig sabihin hindi ang gobyerno ang nagkaloob ng paglilingkod. Ang paglilingkod sa mga pribadong kompanya bilang sekretarya, security guard at iba pa ay halimbawa ng pampribadong paglilingkod.

Samantala, ang kabaliktaran naman ng pampribadong paglilingkod ay ang pampublikong paglilingkod. Ang pampublikong paglilingkod ay ang lahat ng paglilingkod na ibinigay ng gobyerno. Ibig sabihin, hindi pribadong sektor ang nagkaloob ng paglilingkod kung hindi ay ang gobyerno. Ang halimbawa naman ng pampublikong paglilingkod ay ang paglilingkod sa mga departamento o kagawaran ng pamahalaan o gobyerno. Halimbawa ang mga guro na nagtatrabaho sa mga pampublikong paaralan, mga pulis sa PNP, at maging mga sekretarya sa mga kagawaran ng gobyerno ay halimbawa ng pampublikong paglilingkod.
