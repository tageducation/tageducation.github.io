18. Xandria rides through a jeepney which charges ₱ 9.00 for the first 4 kilometers and additional ₱0.50 for each additional kilometer. Express the jeepney fare (F) as function of the number of kilometers (d) that Xandria pays for the ride. with solution po​


PIECEWISE FUNCTION APPLICATIONAnswer with Step-by-step explanation:

The function F that will represent the total jeepney fare that Xandria will pay after d kilometers can be expressed in a piecewise function:

[tex]F(d) =\left \{ {{9.00, d \geq } \atop {9.00 + 0.50(d-4), d > 4 } \right.[/tex]

In the first line of the piecewise function, it represents the value of F(d) that Xandria will pay when d is equal to or less than 4 kilometers. The second line represents the value of F(d) that Xandria will pay for each additional kilometer after the first 4 kilometers.

For example, if Xandria rides the jeepney for 4 kilometers, she will pay 9 pesos only. But when it exceeds the 4 kilometer limit, an additional charge will be added to the total fare. If she rode 6 kilometers, we will use the equation in the second line of the piecewise.

So: F(d) = 9.00 + 0.50(d-4)

F(d) = 9.00 + 0.50(6-4)

F(d) = 9.00 + 0.50(2)

F(d) = 9.00 + 1.00

F(d) = 10.00

Hence, Xandria will pay 10 pesos for her jeepney fare.

Php. 8 pesos for first 4 km.

Php. 0.5 for each succeeding km

Problem:

Function of fare (F) in terms of d (distance)

Solution:

Let d stands for distance in km, and F stands for total fare

F = rate of fare × distance

rate = 8 for 4 km, and 0.5 for each km

rate = 8, if x ≤ 4,

rate = 0.5, if (x-4), and x > 4

Equation:

F = 8 pesos, if x ≤ 4,

F = 8 + 0.5(x-4), if x > 4 (simplify)

F = 8 + 0.5x - 2

F = 0.5x + 6

Therefore, F = 0.5x + 6 is the function of fare in terms of d (distance in km)
