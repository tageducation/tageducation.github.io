14. The given number is 14 a² + 9a—16. if the number 7a²–20a+12 is subtracted from the given number, what number is left?


Answer:

7a² + 29a - 28

Step-by-step explanation:

If we wish to solve this we must convert the English sentence to a mathematical sentence.

"The given number is 14 a² + 9a—16. if the number 7a²–20a+12 is subtracted from the given number"

(14 a² + 9a - 16) - (7a² - 20a + 12)

(14a² + 9a - 16) + (-7a² + 20a - 12)

7a² + 29a - 28
