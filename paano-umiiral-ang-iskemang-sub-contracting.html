2. Paano umiiral ang iskemang sub-contracting


ANG ISKEMANG SUB-CONTRACTING

Ang sub-contracting ay isang kaayusan sa paggawa kung saan kumukontrata ang isang kompanya ng isang ahensiya o indibidwal ay komukontrata ng isang ahensiya o indibidwal na subcontractor upang gawin ang isang trabaho o serbisyo sa isang takdang panahon. Dalawang umiiral na anyo ng subcontracting:

Ang Labor-only Contracting tumutukoy sa pagakaktaon kung saan walang sapat na puhunan upang gawin ang trabaho o serbisyo ang sub-contractor at ang manggagawa na ipinasok ay may direktang kinalaman sa mga gawain ng kompaya.

Ang job-contracting naman ay ang labaliktaran ng labor-only contracting. Ibig sabihin may sapat na puhunan ang sub-contractor para maisagawa ang trabaho at mga gawain ng mga manggagawang ipinasok ng subcontractor. Ang job-contracting ay hindi pinahihintulutan ng batas dahil naaapektuhan nito ang seguridad ng mga manggagawa sa trabaho.

3. Paano umiiral ang iskemang sub contracting


Umiiral ito sa pamamagitan ng dalawang Iskema:

(1)  Labor-only contracting.  Ito ang grupo ng mga taong manggagawa na walang sapat na pondo para sa pangongontrata anupat sila mismo ang gagawa ng mga gawain (labor) para sa isang kliente.

(2) Job contracting.  Ito naman ang grupo ng mga taong nangongontrata ng mga gawain.  Bukod sa sapat ang kanilang pondo, mayroon din silang kasanayan at mga kaukolang permiso para sa mga uri ng trabaho.

Umiiral ang iskemang ito dahil mas makakatipid ang mga kompanya kung iba ang gagawa ng trabaho para sa kanila at hindi sila mismo ang maghahanap ng mga taong magiging empleyado nila.  Isa pa, ang mga trabahong idinadaan sa kontrata ay hindi panghabambuhay kundi mayroong yugto ng panahon lamang.

Sistemang subcontracting

Ang subcontracting ay isang uri ng sistema na kung saan ang isang kumpanya ay kumukuha ng mga manggagawa mula sa ibang kumpanya o grupo para gumawa ng isang proyekto.  

Bakit may subcontracting? Ang pangunahing dahilan ng subcontracting ay ang mababang halaga ng paggawa ng proyekto Mas mapadadali ang mga malalaking proyekto sa paggamit ng subcontracting Walang pang matagalan na obligasyon ang mga kumpanya sa isa't isa

Ang mga self employed without any pay o ang unpaid family labor ay madalas na mga manggagawa na hindi nakatatanggap ng sapat na kita o hindi kaya ay binabayaran sa kanilang serbisyo.

Self-employed

1) Tubero

2) Electrician

3) Mekaniko

4) Online Freelancer

5) Hardinero

18. Ano ang iskemang subcontracting?? Ano ang labor only contracting??


Iskemang Subcontracting

Ang iskemang subcontracting ay isang termino na ginagamit sa larangan ng negosyo na kung saan ang mga kumpanya ay kumukuha ng mga manggagawa o subcontractor para gampanan ang mga tungkulin o trabahong kinakailangan ng isang kumpanya. Halimbawa nito ay ang mga sumusunod:

Paghire ng mga kumpanya ng mga electricians Paghire ng mga kumpanya ng mga construction workers sa pagpapatayo ng mga gusali Paghire ng mga kumpanya ng mga plumbers para sa itinatayong kumpanya

Labor-only Contracting

Ang labor-only contracting ay isang termino na ginagamit naman upang ilarawan ang paguha ng mga subcontractor ng mga manggagawa para sa isang kumpanya.
