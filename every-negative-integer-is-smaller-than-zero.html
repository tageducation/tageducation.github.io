Numbers

1. The smallest negative integer is -1. Wrong statement.

2. The largest negative integer is - 1 000. Wrong statement.

3. All numbers greater than zero are positive integers. Wrong statement.

4. The repeating decimal 0.1616... is a rational number. True statement.

5. The number pi is an irrational number. True statement.

6. Every negative integer is less than zero. True statement.

7. Irrational numbers can be found on the number line. True statement.

8. Decimal numbers are whole numbers. Wrong statement.

9. Natural numbers are irrational. Wrong statement.

10. Zero is a natural number. Wrong statement.

Let's discuss one by one the answer to the true or false statement of the 10 questions above.

1. Integers consist of whole numbers and negative integers. The number of membership is unlimited. (…,-5,-4,-3,-2,-1,0,1,2,3,4,…). So the smallest negative integer is not -1 but infinity because -2<-1, -100<-1, which is true is the largest integer.

2. The largest negative integer is - 1 000, which is a false statement. The explanation is the same as number 1.

3. All numbers greater than zero are positive integers. This too is a false statement. Why? [tex]\frac{1}{2}[/tex]>0 but [tex]\frac{1}{2}[/tex] is not an integer, it is a fraction. 2>0 but 2 is not an integer. So not every number greater than 0 is a positive integer.

4. The repeating decimal 0.1616... is a rational number. This is a true statement, because one of the characteristics of rational numbers is that the number can be converted into an ordinary fraction (a/b) and if this number is converted to a decimal fraction, the number will stop at a certain number. If it does not stop, it will form a repeating pattern. And 0.1616… is a form of repetition pattern.

Let's find the fraction form 0.1616..

Multiply 0.1616 by 10=1.6161…

Multiply by 1000=161,6161…

1000-10=161.6161-1.6161.

990=160.

99=16=16/99.

5. The number pi (Π) is an irrational number. This statement is true because the value of =3.141592653589… the number after the comma does not stop at a number.

6. Every negative integer is less than zero. This is a true statement, because all negative numbers are to the left of 0 on the number line. (…,-5,-4,-3,-2,-1,0,1,2,3,4,…).

7. Irrational numbers can be found on the number line. This is a true statement because irrational numbers are real numbers. And all real numbers can be found on the number line.

8. Decimal numbers are whole numbers. Wrong statement. Because the whole number consists of natural integers, zero, and negative integers. Therefore, decimal numbers are not included in the set of integers.

9. Natural numbers are irrational. This is a false statement. Because natural numbers start from 1,2,3,4,5,… while irrational numbers are numbers that cannot be expressed as a division of two integers. That is, rational numbers are real numbers that cannot be divided.

10. Zero is a natural number. This is a wrong statement because as explained above natural numbers = {1,2,3,4...} while 0 is not included in the membership of natural numbers but may be included in the membership of whole numbers.
