WHAT IS Ogive?

An Ogive is a type of line graph that shows cumulative frequencies. The x-axis has the class boundaries of the class intervals while the y-axis are the possible partial sums of frequencies or cumulative frequency.

Using the same data set above, we need to complete the other details for the Ogive – the class boundaries and the corresponding cumulative frequency of each interval.(See in the first picture).

Try to answer the following using the Ogive at the second picture.

1. How many students have a grade of lower than 80?

We select the boundary 79.5 because it is lower than 80. Based on the Ogive, the 79.5 has a plot of 14, thus there are 14 students with grades lower than 80. We cannot select 84.5 because it is more than 80.

2. How many students have a grade of AT MOST 90?

We select the boundary 89.5 because we need grades that are 90 or lower. The plot of 89.5 is 36. Therefore, the answer is 36 students who have a grade of at most 90.

12. There were 187 passengers in a bus. There were 37 more adult passengers than children passengers. If there were 20 more adult male passengers than adult femalepassengers on the bus, how many females were there?​


Answer:

There are 46 female passengers

Step-by-step explanation:

Let A: Adult(male)

Let B: Adult(Female)

Let C: Children

Total Passengers = A+B+C

Eq. 1 : 187= A + B + C

Eq. 2: A + B= C + 37

Eq. 3: A= B + 20

Substitute eq. 3 to eq. 2

A + B = C + 37

B + 20 + B= C + 37

2B + 20 - 37 = C

C= 2B - 17

Substitute C and A to eq. 1

187= A + B + C

187= (B+20) + B + (2B-17)

187=4B+3

187-3=4B

B= 46
