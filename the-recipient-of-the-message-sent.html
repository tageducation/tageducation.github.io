21. 9. Which communication model focuses on the message sent to the recipient?A. Interactive ModelC. Transactional modelB. Linear ModelD. A combination of A and B​


Communication in its development and use involves various communication models that are adapted to their use. The communication model can be interpreted as a description of the classification of an activity that can be seen from the point of view of the sender of the message, the message itself, the media used, and from the point of view of the recipient of the message.

Linear communication is a process of delivering messages from a communicator to the communicant directly or indirectly using various communication media. Where this communication only occurs in one direction without reciprocity of the message conveyed by the communicator. So in short it can be said that linear communication is one-way communication.

Claude Shannon and Warren Weaver describe communication as a linear process because they are interested in radio and telephone technology and want to develop a model that can explain how to pass through various channels (channels). Furthermore, Claude Shannon and Warren Weaver explain that the linear model assumes that a person is only a sender or receiver. Of course this is a very narrow view of the participants in the communication process. An important concept in this model is interference (noise), which is any additional stimulus that cannot interfere with the accuracy of the message conveyed. This interference is always on the line with a message received by the recipient.

The need for this communication requires the use of a communication model that is in accordance with various existing situations and conditions, one of which is a linear communication model, where this communication model is a one-way communication whose use is usually used in various instructional activities or explanations that do not occur reciprocally.

As an example of this linear communication we often encounter in daily activities, for example, communication on television, radio and various kinds of writings in the mass media. All of these things are linear aka one-way communication, because we as communicants do not give reciprocity. Another example is greeting or greeting in various events, this is also an example of linear communication.
