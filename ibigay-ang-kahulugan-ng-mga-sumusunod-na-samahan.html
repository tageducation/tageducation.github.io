1. ibigay ang kahulugan ng mga sumusunod na samahan ​


Ang isang samahan ay isang pangkat ng mga tao na nagtutulungan, tulad ng isang samahan ng kapitbahayan, isang charity, isang unyon, o isang korporasyon. Ang samahan ay ang kilos ng pagbubuo o pagtataguyod ng isang bagay (tulad ng isang samahan). Maaari rin itong mag-refer sa isang sistema ng pag-aayos o pagkakasunud-sunod, o isang istraktura para sa pag-uuri ng mga bagay.
