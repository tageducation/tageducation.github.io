28. BEAR is a trapezoid with segment TS as its median if ts = 5x + 3 and be = 4x + 1 and ra= 8 x -3 find ​, TS, BE and AR


Median of Trapezoid is given as,

[tex]\sf median = \frac{(b_1 + \;b_2)}{2} \\\\where\\\\median = 5x + 3\\\\b_1 = 4x + 1\\\\b_2 = 8x - 3\\\\\text{\sf solve for x,}\\\\median = \frac{(b_1 + \;b_2)}{2} \\\\5x + 3 = \frac{(4x + 1) + (8x - 3)}{2} \\\\2(5x + 3) = (4x + 1) + (8x - 3)\\\\10x + 6 = 12x - 2\\\\12x - 10x = 6 + 2\\\\2x = 8\\\\x = \frac{8}{2} \\\\x=4[/tex]

Solve for length of base1,

base1 = 4x + 1

base1 = 4(4) + 1

base1 = 16 + 1

base1 = 17

Solve for length of base2,

base2 = 8x - 3

base2 = 8(4) - 3

base2 = 32 - 3

base2 = 29

Solve for length of median,

median = 5x + 3

median = 5(4) + 3

median = 20 + 3

median = 23
