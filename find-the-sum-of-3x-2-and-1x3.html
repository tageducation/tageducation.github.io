Step-by-step explanation:

To find the sum of rational expressions, we must first make sure the expressions have a common denominator. In this case, the common denominator is $(x-1)(x+2)$. We can then add the expressions by adding the numerators after multiplying each expression by the appropriate factor to make the denominators equal the common denominator.

The first expression is already in the desired form, so we do not need to change it. The second expression can be rewritten as $\frac{3(x+2)}{(x-1)(x+2)}$. Thus, the sum of the two expressions is

$$\frac{1}{x+2} + \frac{3(x+2)}{(x-1)(x+2)} = \frac{1(x-1)(x+2) + 3(x+2)}{(x-1)(x+2)} = \frac{(x-1)(x+2) + 3(x+2)}{(x-1)(x+2)} = \frac{(x+2)(x-1 + 3)}{(x-1)(x+2)} = \frac{(x+2)(x+2)}{(x-1)(x+2)} = \frac{x+2}{x-1}.$$

Therefore, the sum of the given expressions is $\frac{x+2}{x-1}$.
