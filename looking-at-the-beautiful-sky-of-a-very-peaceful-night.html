The polygon that has a least number of sides is triangle (3). Hence, consider n-gons that satisfy the inequality: 3 ≤ n ≤ 6.

Then, add the combinations of each n-gons on the inequality 3 ≤ n ≤ 6 to find the number of polygons that can be formed, which gives us

⁶C₃ + ⁶C₄ + ⁶C₅ + ⁶C₆

= 6!/[3!(6 - 3)!] + 6!/[4!(6 - 4)!] + 6!/[5!(6 - 5)!] + 6!/[6!(6 - 6)!]

= 20 + 15 + 6 + 1

= 42
