26. characteristics of network interface card​


Answer:

• NIC stands for Network Interface Card.

• NIC act as the physical interface between the computer and the network cable.

• NIC is one of the hardware device present on the motherboard(i.e. on the PCI slot)

• PCI stands for Peripheral Component Interconnect

• NIC also called as Network Adapter Card

• Each computer in computer network must have a card called as NIC

• Computer can communicate with computer network by using NIC

• NIC card perform all network related functions on the behalf of the computer.

Functions of NIC:

• Prepare data from the computer for the network cable.

• Send the data to another computer

• Control the flow of data between the computers and the cabling

• Receive incoming data from the cable and translate it into bytes that can be understood by the computers CPU.

27. function of network interface card​


The NIC contains the electronic circuitry required to communicate using a wired connection (e.g., Ethernet) or a wireless connection (e.g., WiFi). A network interface card is also known as a network interface controller, network adapter, or Local Area Network (LAN) adapter.


28. what is the function of network interface card​


The main function of Network Interface Card or NIC is connect the computer to the network.  

Network Interface Card is a part of the computer which is normally attached to the motherboard by a PCI slot. For built-in motherboards, it is already on the inside. It is often known as LAN Card (Local Area Network) or Ethernet Card. NIC has a slot, where cables could be attached. But if it's for wireless devices, it has antennas or blinking light indicator. It is normally located at the back of the CPU, near USB ports and it requires drivers to install in your computer.

NIC works by reading the in the CPU then transmit it to the destination device. Transmission could be via frequencies (for wireless network) and could be data travel by cables (if wired). Then once the data was sent, it will send an acknowledge or request back to the computer.
