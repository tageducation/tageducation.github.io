7. In your own words, how do you represent rational function using: a. Equation; b. Graph; and c. Table of values. ​


Answer:

A rational function is a type of function that is in the form of [tex]f(x)=\frac{P(x)}{Q(x)}[/tex] . Both [tex]P(x)[/tex] and [tex]Q(x)[/tex] are also functions with a restriction that [tex]Q(x) \ne 0\\[/tex].

Explanation:

Representing A Function Using an Equation

Solving problems that involve variation is one example where we can represent a rational function using an equation.  

Indirect variation is a relationship of two variables where one of the variables increases and the other one decreases. Thus, they go in the opposite direction.  

We can say that [tex]y[/tex] varies inversely as [tex]x[/tex] then there exists a constant [tex]k[/tex], such that [tex]y=\frac{k}{x}[/tex]. We just presented a rational function using an equation.  

Representing A Function Using a Table of Values and Graphs

Let's take the function [tex]f(x)=\frac{x+1}{x}[/tex]. Now since 0 can't be a value of [tex]x[/tex], pick random integers that may assume as our x values. Substitute each value of

[tex]\begin{matrix}x & -4 & -3 & -2 &-1&-0.5&1&2&3\\f(x) & 0.75 & 0.67 & 0.5&0&-1&2&1.5&1.33\end{matrix}[/tex]

Above is the table of values representing the function.
